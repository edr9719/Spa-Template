# Galería de Imágenes

Este directorio contendrá las fotos reales del spa una vez que estén disponibles.

## Fotos Recomendadas (8-12 imágenes):
- Masajes: 2-3 fotos de sesiones de masaje y sala de masajes
- Faciales: 1-2 fotos de tratamientos faciales
- Manicura/Pedicura: 1-2 fotos de estaciones y servicios
- Instalaciones: 3-4 fotos del ambiente, recepción, áreas de relajación
- Productos: 1 foto de productos premium utilizados

## Especificaciones Técnicas:
- Formato: JPG o PNG
- Tamaño recomendado: 1200x1200px (cuadrado)
- Peso máximo: 500KB por imagen
- Nombres sugeridos: masaje-1.jpg, facial-1.jpg, manicura-1.jpg, etc.
